## Getting Started

To get a local copy up and running follow these simple example steps.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/harisem/dans-multi-pro-test.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Copy the .env file
   ```sh
   cp .env.example .env
   ```
4. Start development server
    ```sh
    npm run start
    ```
